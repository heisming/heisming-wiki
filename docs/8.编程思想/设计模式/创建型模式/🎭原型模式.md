## JS的原型链
此处应该引用到JS基础的原型链知识

> class是function的语法糖, 举个例子
```ts
class Foo {
  name: string;
  age: number;
  constructor(name: string, age: number) {
    this.name = name;
    this.age = age;
  }
  getName() {
    return this.name;
  }
}
const f1 = new Foo('liming', 28);
console.log(Foo.prototype)
console.log(f1.__proto__);
console.log(f1.__proto__ === Foo.prototype)
```
